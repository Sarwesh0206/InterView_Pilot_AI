Accenture Recruitment Process Overview
The Accenture recruitment process for freshers typically includes four to five stages:

Application and Screening

Aptitude Test (Written Test)

Coding Round

Technical Interview

HR Interview

Some drives include an additional Group Discussion or Assessment Task stage.

Stage 1: Application and Screening
Register on Accenture careers portal or through college placement cell.

Upload resume, academic records, and skills.

Screening based on GPA, backlogs, and skill match.

Stage 2: Aptitude Test
Online test with 40–60 questions.

Duration: 40–60 minutes.

Sections: Quantitative Aptitude, Logical Reasoning, Verbal Ability.

Stage 3: Coding Round
Online coding test with 1–3 problems.

Duration: 40–60 minutes.

Languages: C, C++, Java, Python.

Topics: Arrays, Strings, Loops, Conditions, Basic Algorithms.

Stage 4: Technical Interview
20–40 minutes one-on-one interview.

Topics: Programming fundamentals, OOP, DBMS, OS, Networks, Projects.

Stage 5: HR Interview
10–20 minutes interview.

Topics: Communication, motivation, career goals, salary expectations.

Sample Coding Questions
Q1. Find the maximum element in an array.

python
def max_element(arr):
    return max(arr)
Q2. Check if a string is a palindrome.

python
def is_palindrome(s):
    return s == s[::-1]
Q3. Print Fibonacci series up to n terms.

python
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        print(a, end=' ')
        a, b = b, a + b
Q4. Find factorial of a number.

python
def factorial(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial(n - 1)
Q5. Reverse a string.

python
def reverse_string(s):
    return s[::-1]

Sample Technical Questions and Answers
Q1. What is OOP? Explain its features.
Answer: OOP (Object-Oriented Programming) is a paradigm based on objects containing data and methods. Features: Class, Object, Inheritance, Polymorphism, Abstraction, Encapsulation.

Q2. What is the difference between SQL and NoSQL?
Answer: SQL is relational, structured, uses tables. NoSQL is non-relational, unstructured, uses documents, graphs, or key-value stores.

Q3. Explain process and thread.
Answer: Process is an independent program execution. Thread is a lightweight unit within a process. Multiple threads can exist in one process.

Q4. What is a primary key and foreign key?
Answer: Primary key uniquely identifies a record. Foreign key links two tables, enforcing referential integrity.

Q5. Write a SQL query to find the second highest salary.

sql
SELECT MAX(Salary) AS SecondHighest
FROM Employee
WHERE Salary < (SELECT MAX(Salary) FROM Employee);
Q6. What is recursion?
Answer: Recursion is a function calling itself to solve a problem. Example: factorial, Fibonacci.

Q7. Explain TCP and UDP.
Answer: TCP is connection-oriented, reliable. UDP is connection-less, faster but unreliable.

Q8. What is your favorite project? Explain.
Answer: Describe project goal, technologies used, challenges, and results. Example: ML dashboard using Python, Streamlit, FastAPI.

Accenture HR Interview – Questions and Sample Answers
Interview Duration
10–20 minutes.

Focus on communication, motivation, and fit.

Common HR Questions and Answers
Q1. Tell me about yourself.
Answer: I am a third-year B.Tech student in Electronics and Instrumentation Engineering. I have strong interest in machine learning, full-stack development, and data analytics. I have worked on coding projects using Python, FastAPI, and Streamlit, and I actively participate in college hackathons.

Q2. What are your strengths and weaknesses?
Answer: Strengths: problem-solving, quick learning, teamwork. Weakness: I sometimes take time to speak in group discussions. I am improving by practicing active participation.

Q3. Why should we hire you?
Answer: I have strong technical skills in Python, machine learning, and full-stack development, matching role requirements. I have completed projects using FastAPI and Streamlit, and I am eager to learn and contribute.

Q4. Why do you want to work at Accenture?
Answer: Accenture works on innovative digital transformation and cloud solutions, aligning with my interest in AI and data. The role offers opportunities to work on real projects and learn from experienced teams.

Q5. What are your career goals?
Answer: Short-term: gain hands-on experience in ML and data analytics. Long-term: become a data scientist leading impactful projects.

Q6. How do you handle stress?
Answer: I prioritize tasks, break them into smaller steps, and take short breaks. During a hackathon, I stayed calm and resolved a critical bug under time pressure.

Q7. What are your salary expectations?
Answer: As a fresher, I focus on learning. I expect a salary aligned with industry standards and open to discussion based on company policies.
