Sample Quantitative Questions
Q1. What is 20% of 250?
Answer: 50

Q2. If CP = Rs. 500, SP = Rs. 600, find profit %.
Answer: Profit = 100, Profit % = (100/500) × 100 = 20%

Q3. A can do work in 12 days, B in 18 days. Together?
Answer: Together = (12×18)/(12+18) = 216/30 = 7.2 days

Q4. Car travels 150 km in 3 hours. Speed?
Answer: 150/3 = 50 km/h

Q5. Sum of first 10 natural numbers?
Answer: n(n+1)/2 = 10×11/2 = 55

Q3. Find Factorial of a Number

python
def factorial(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial(n - 1)
Q4. Print Fibonacci Series up to n Terms

python
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        print(a, end=' ')
        a, b = b, a + b
Q5. Binary Search

python
def binary_search(arr, x):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == x:
            return mid
        elif arr[mid] < x:
            left = mid + 1
        else:
            right = mid - 1
    return -1
Q6. Combination Sum

python
def combination_sum(candidates, target):
    result = []
    def backtrack(start, path, total):
        if total == target:
            result.append(path[:])
            return
        if total > target:
            return
        for i in range(start, len(candidates)):
            backtrack(i, path + [candidates[i]], total + candidates[i])
    backtrack(0, [], 0)
    return result

Sample Technical Questions and Answers
Q1. What is OOP? Explain its features.
Answer: OOP (Object-Oriented Programming) is a paradigm based on objects containing data and methods. Features: Class, Object, Inheritance, Polymorphism, Abstraction, Encapsulation.

Q2. What is the difference between SQL and NoSQL?
Answer: SQL is relational, structured, uses tables. NoSQL is non-relational, unstructured, uses documents, graphs, or key-value stores.

Q3. Explain process and thread.
Answer: Process is an independent program execution. Thread is a lightweight unit within a process. Multiple threads can exist in one process.

Q4. What is a primary key and foreign key?
Answer: Primary key uniquely identifies a record. Foreign key links two tables, enforcing referential integrity.

Q5. Write a SQL query to find the second highest salary.

sql
SELECT MAX(Salary) AS SecondHighest
FROM Employee
WHERE Salary < (SELECT MAX(Salary) FROM Employee);
Q6. What is recursion? Give an example.
Answer: Recursion is a function calling itself to solve a problem. Example: factorial, Fibonacci.

Q7. Explain TCP and UDP.
Answer: TCP is connection-oriented, reliable. UDP is connection-less, faster but unreliable.

Q8. What is your favorite project? Explain goal, tech, challenges, results.
Answer: Example: ML dashboard using Python, Streamlit, FastAPI. Goal: visualize data. Tech: Pandas, Streamlit. Challenge: data cleaning. Result: accurate dashboard.

Q9. What is stack overflow?
Answer: Stack overflow occurs when a program exceeds stack memory, often due to infinite recursion or large local variables.

Q10. Difference between array and list.
Answer: Array: fixed size, same type. List: dynamic size, can hold mixed types (in Python).
