Wipro Hiring Process Overview
The Wipro recruitment process for freshers typically includes 3–4 stages over 1–3 weeks:

Registration (Wipro NLTH / Turbo / Elite / Campus Drive)

Online Assessment (Aptitude + Technical MCQs + Coding + Essay Writing)

Technical Interview

HR Interview

Some drives combine Technical and HR into one round. The process is hosted on AMCAT (for NLTH) or Superset platform.

Stage 1: Registration (Wipro NLTH / Turbo / Elite)
Register on Wipro careers portal or via college placement cell.

Wipro NLTH: for Project Engineer role (aptitude-heavy, 60% filter).

Turbo/Elite: for Software Engineer/Analyst roles (coding-heavy, higher salary).

Upload resume, academic records, skills (C/C++/Java/Python preferred).

Stage 2: Online Assessment
Duration: 128 minutes (no sectional timing, no negative marking).

Sections:

Aptitude Test (48 questions, 48 mins): Quantitative (20), Logical (20), Verbal (8).

Technical MCQs (12 questions, 12 mins): Code output prediction, problem identification, time complexity, OS/DBMS/CN basics.

Coding Test (2 questions, 60 mins): Simple array/string problems (C++/Java only).

Essay Writing (8 mins): Communication assessment.

Stage 3: Technical Interview
Duration: 20–30 minutes, one-on-one.

Topics: OS (deadlocks, semaphores), DBMS (joins, queries), OOPs, DSA (basic: arrays, strings, sorting), projects/resume deep dive.

Expect to write code on whiteboard or shared editor.

Stage 4: HR Interview
Duration: 10–20 minutes.

Topics: Communication, motivation, career goals, salary expectations, behavioral questions (STAR method).

Often combined with Technical Interview in some drives.

Sample Coding Questions (Wipro SDE Sheet)
Q1. Reverse an Array

cpp
#include <iostream>
using namespace std;
void reverse_array(int arr[], int n) {
    for (int i = 0; i < n / 2; i++) {
        swap(arr[i], arr[n - 1 - i]);
    }
}
Q2. Check if a String is Palindrome

java
public class Palindrome {
    public static boolean isPalindrome(String s) {
        return s.equals(new StringBuilder(s).reverse().toString());
    }
}
Q3. Find GCD of Two Numbers

cpp
int gcd(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
Q4. Print First n Fibonacci Numbers

java
public class Fibonacci {
    public static void printFibonacci(int n) {
        int a = 0, b = 1;
        for (int i = 0; i < n; i++) {
            System.out.print(a + " ");
            int temp = a + b;
            a = b;
            b = temp;
        }
    }
}
Q5. Bubble Sort

cpp
void bubble_sort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                swap(arr[j], arr[j + 1]);
            }
        }
    }
}
Q6. Print Perfect Numbers up to n

cpp
bool is_perfect(int num) {
    int sum = 1;
    for (int i = 2; i * i <= num; i++) {
        if (num % i == 0) {
            sum += i;
            if (i * i != num) sum += num / i;
        }
    }
    return sum == num;
}

Sample Technical Questions and Answers
Q1. What is OOP? Explain its features.
Answer: OOP (Object-Oriented Programming) is a paradigm based on objects containing data and methods. Features: Class, Object, Inheritance, Polymorphism, Abstraction, Encapsulation.

Q2. What is the difference between SQL and NoSQL?
Answer: SQL is relational, structured, uses tables. NoSQL is non-relational, unstructured, uses documents, graphs, or key-value stores.

Q3. Explain deadlock and semaphores.
Answer: Deadlock: processes wait indefinitely for resources. Semaphores: synchronization primitives to control access to resources.

Q4. What is a join? Types of joins?
Answer: Join combines rows from two tables. Types: INNER, LEFT, RIGHT, FULL.

Q5. Write a SQL query to find the second highest salary.

sql
SELECT MAX(Salary) AS SecondHighest
FROM Employee
WHERE Salary < (SELECT MAX(Salary) FROM Employee);
Q6. What is recursion? Give an example.
Answer: Recursion is a function calling itself. Example: factorial, Fibonacci.

Q7. Explain TCP and UDP.
Answer: TCP is connection-oriented, reliable. UDP is connection-less, faster but unreliable.

Q8. What is your favorite project? Explain goal, tech, challenges, results.
Answer: Example: ML dashboard using Python, Streamlit, FastAPI. Goal: visualize data. Tech: Pandas, Streamlit. Challenge: data cleaning. Result: accurate dashboard.

Q9. What is bubble sort? Time complexity?
Answer: Bubble sort swaps adjacent elements if out of order. Time complexity: O(n²).

Q10. Difference between array and linked list.
Answer: Array: fixed size, contiguous memory. Linked list: dynamic size, non-contiguous, uses pointers.
